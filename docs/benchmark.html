🎬 The streaming self-improvement protocol

🎬clip stream
→
1👁️Watch → write skill codeVLM reads clip · LLM emits code-as-policy
→
2🔬Sim execution (F)run on LIBERO/Robosuite/ManiSkill · traces
→
3📚Skill library (M)store validated skills · retrieve for next clip
→
📈compounding curve
↺ per clip: refine the library from feedback · measure held-out success vs. # clips seen
Figure 1 · A self-improving agent's held-out curve rises with clips watched; a zero-shot or VLA agent's stays flat. That separation is the benchmark's validity — and the paper's headline figure.
